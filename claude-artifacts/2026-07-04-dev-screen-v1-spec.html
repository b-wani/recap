개발 데모 녹화 앱 — v1 스펙

제품 정의

웹 프론트엔드 개발자가 개발 화면을 녹화하고, 자동 줌과 커서 효과를 입혀 GitHub PR/이슈와 업무 티켓에 첨부할 1~2분 데모 영상을 만드는 macOS 개인 도구입니다.

Screen Studio와 Cap의 공통 코어 기능만 벤치마킹합니다.

02

아키텍처 — 후처리 렌더링 모델

녹화 중에는 효과를 굽지 않습니다. 원본 영상과 이벤트 트랙(마우스 위치·클릭·커서 타입·키 타임스탬프)만 기록합니다.

자동 효과는 이벤트 트랙에서 렌더 레시피로 생성되고, 익스포트 때 원본에 합성됩니다. 편집이란 이 레시피를 고치는 일이며, 원본은 불변입니다.

실선 = 데이터 흐름. 점선 = 편집 루프(레시피 수정 → 미리보기 반영 → 익스포트 입력). 효과 렌더링은 전부 TypeScript(Chromium) 영역에 있고, Swift 사이드카는 기록만 담당하는 불변층입니다.

03

자동 줌 규칙

클릭 하나의 수명주기로 정리한 규칙입니다. 이 규칙이 이벤트 트랙에서 줌 구간을 자동 생성하는 알고리즘의 스펙입니다.

클릭 −0.5초

선행 줌인

클릭보다 0.5초 앞서 부드럽게 확대를 시작합니다. 시청자는 클릭 순간을 이미 확대된 화면에서 봅니다. 중심은 클릭 좌표이며, 화면 가장자리에서는 프레임이 원본 밖으로 나가지 않도록 중심을 보정합니다.

트리거: 클릭만배율: 전역 2.0x (1.5/2/2.5)
유지 중

병합과 팬

3초 이내에 다음 클릭이 이어지면 줌을 유지합니다. 새 클릭이 현재 뷰 밖이면 줌아웃 대신 카메라를 이동(팬)합니다. 키 입력은 줌을 새로 만들지 않지만 "활동"으로 카운트되어, 타이핑하는 동안 줌이 풀리지 않습니다.

병합 임계 3초키 입력 = 유지 조건
활동 +2초

줌아웃

마지막 활동 후 2초간 아무 일도 없으면 부드럽게 전체 화면으로 돌아갑니다.

편집 시

사용자 보정

자동 생성된 줌 구간은 에디터 타임라인에 표시되며, 삭제·이동·길이 조절이 가능합니다. 자동 줌이 오판해도 재녹화 없이 구간 하나만 고치면 됩니다.

04

커서 렌더링

시스템 커서는 캡처에서 제외하고, 이벤트 트랙의 좌표로 벡터 커서를 다시 그립니다. 그래서 녹화가 끝난 뒤에도 크기와 스무딩을 바꿀 수 있습니다.

스무딩

원시 좌표를 스플라인 보간으로 부드러운 곡선 이동으로 변환합니다.

끔약강

크기

실제보다 크게 그려 데모 가독성을 높입니다.

1x1.5x2x

클릭 효과

클릭 순간 퍼지는 리플(원)과 살짝 눌리는 스케일 애니메이션을 넣습니다.

커서 모양 3종

화살표 · 포인터(손가락) · I-beam을 기록해 재현합니다. hover 시 포인터로 바뀌는 신호가 살아납니다. 그 외 커서는 화살표로 대체합니다.

05

저장 모델

프로젝트는 평문 JSON과 표준 영상 포맷으로 된 폴더 번들입니다. 앱이 없어도 QuickTime과 텍스트 에디터로 열립니다. 자동 삭제는 없으며, 정리는 폴더 단위로 직접 합니다.

~/Movies/DevScreen/2026-07-04_1432/
raw.mp4          # 원본 영상 (60fps · Retina 2x · 커서 없음)
events.json      # 이벤트 트랙 — 마우스 위치·클릭·커서 타입·키 타임스탬프
recipe.json      # 렌더 레시피 — 줌 구간·트림·스타일·커서 설정
export-1.mp4     # 익스포트 산출물 (같은 폴더에 누적)
export-2.gif

에디터의 "최근 녹화" 목록에서 아무 프로젝트나 다시 열어 레시피를 수정하고 재익스포트할 수 있습니다.

06

사용 동선

"개발 중 → 녹화 → 확인 → 붙여넣기"의 마찰을 최소화합니다. 메뉴바 상주 앱이며 Dock 아이콘은 없습니다.

시작

윈도우 선택 → 즉시 녹화

메뉴바 패널에서 브라우저 창을 고르면 바로 녹화가 시작됩니다. 카운트다운은 없습니다 — 준비 동작은 앞 트리밍으로 잘라냅니다. 창 크기는 시작 시점에 고정됩니다.

⌥⌘R 토글녹화 중: 빨간 점 + 경과 시간
정지

에디터 창 자동 오픈

미리보기, 경량 편집(트리밍·줌 구간·스타일), 익스포트 버튼이 한 창에 모여 있습니다. 이 창이 최근 녹화 목록의 진입점도 겸합니다.

완료

익스포트 → 클립보드 → ⌘V

익스포트가 끝나면 파일이 클립보드에 자동 복사되고 알림이 뜹니다. PR 코멘트나 티켓 본문에 바로 붙여넣으면 동선이 끝납니다.
