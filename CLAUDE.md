## Agent skills

### Issue tracker

이슈는 GitHub(b-wani/recap) Issues에서 관리한다. 외부 PR은 트리아지 대상이 아니다. See `docs/agents/issue-tracker.md`.

트리아지 라벨은 canonical 기본값(needs-triage, needs-info, ready-for-agent, ready-for-human, wontfix)을 그대로 사용한다. See `docs/agents/triage-labels.md`.
