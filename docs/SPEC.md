# Recap v1 스펙

> 2026-07-04 그릴링 세션에서 확정. 용어는 [CONTEXT.md](../CONTEXT.md), 스택 결정 배경은 [ADR 0001](./adr/0001-electron-with-swift-capture-sidecar.md) 참고.

## 제품 정의

웹 프론트엔드 개발자가 개발 화면을 녹화하고, 자동 줌·커서 효과를 입혀, GitHub PR/이슈와 업무 티켓에 첨부할 1~2분 데모 영상을 만드는 macOS 개인 도구. Screen Studio와 Cap의 공통 코어 기능만 벤치마킹한다.

## 범위

### v1 포함

- 녹화 → 자동 효과 → 미리보기 + 경량 편집 → 익스포트 파이프라인
- 경량 편집 3종: 앞뒤 트리밍, 줌 구간 삭제/이동/길이 조절, 배경/패딩 스타일 조절
- 뷰포트 크기 배지 오버레이
- GitHub 익스포트 프리셋 + 클립보드 복사

### v1 명시적 제외

컷 편집(중간 잘라내기 — 실수하면 재녹화), 속도 조절, 자막/트랜스크립트, 웹캠 오버레이, 오디오(전 구간 무음), 주석 도구, 모션 블러.

### v2 예약

Before/After 비교 렌더링, 키 입력 오버레이, 브랜치/커밋 배지, Dooray 티켓 첨부, 영역 캡처, 스타일 프리셋 저장, 셀렉터 표시(CDP)는 탈락.

## 아키텍처

### 후처리 렌더링 모델

- 녹화 중에는 **원본 영상**과 **이벤트 트랙**(마우스 위치, 클릭, 커서 타입, 키 입력 타임스탬프)만 기록한다. 효과는 굽지 않는다.
- 자동 효과(줌, 커서 스무딩)는 이벤트 트랙에서 **렌더 레시피**를 자동 생성하고, 익스포트 시 원본에 합성한다.
- 편집 = 렌더 레시피 수정. 원본은 불변.

### 스택 (ADR 0001)

- **Electron + TypeScript 본체**: UI, 미리보기, 효과 렌더링, 익스포트. 합성 파이프라인은 Canvas/WebGL + WebCodecs (Chromium 미디어 스택).
- **Swift 캡처 사이드카**: ~200줄 CLI. ScreenCaptureKit으로 원본 기록, 마우스/키 이벤트 스트리밍. 효과 로직을 넣지 않는 불변층.

## 캡처

- **윈도우 캡처만.** 창 하나를 선택해 녹화. 다른 창에 가려져도 무관.
- 창 크기는 녹화 시작 시점에 고정. 녹화 중 리사이즈는 미지원 — 반응형 데모는 DevTools 반응형 모드로 우회.
- 60fps, Retina 2x 해상도로 원본 기록.
- 시스템 커서는 캡처에서 제외 (렌더링 때 다시 그림).

## 자동 줌 규칙

1. **트리거**: 마우스 클릭만. (타이핑·스크롤·호버는 트리거 아님)
2. **타이밍**: 클릭 0.5초 전부터 부드럽게 줌인 시작.
3. **중심**: 클릭 좌표. 프레임이 원본 밖으로 나가지 않도록 가장자리 클램핑.
4. **병합**: 다음 클릭이 3초 이내면 줌 유지. 새 클릭이 현재 뷰 밖이면 팬으로 이동.
5. **해제**: 마지막 활동 2초 후 부드럽게 줌아웃.
6. **배율**: 전역 단일 설정 (1.5x / 2.0x / 2.5x, 기본 2.0x). 구간별 배율은 v2.
7. **키 입력은 유지 조건**: 줌을 트리거하지 않지만 규칙 5의 "활동"으로 카운트 — 타이핑 중 줌이 풀리지 않는다.

## 커서 렌더링

- 이벤트 트랙 좌표로 벡터 커서를 다시 그린다.
- **스무딩**: 스플라인 보간. 전역 설정 끔/약/강.
- **크기**: 전역 설정 1x / 1.5x / 2x.
- **클릭 효과**: 리플(퍼지는 원) + 눌림 스케일 애니메이션.
- **커서 모양 3종** 기록·재현: 화살표, 포인터(손가락), I-beam. 그 외 커서는 화살표로 대체.

## 저장 모델

- 프로젝트 = 폴더 번들: `~/Movies/Recap/{YYYY-MM-DD_HHmm}/`
  - `raw.mp4` — 원본 영상
  - `events.json` — 이벤트 트랙
  - `recipe.json` — 렌더 레시피
  - 익스포트 산출물도 같은 폴더에
- 평문 JSON + 표준 영상 포맷: 앱 없이도 QuickTime/에디터로 열린다.
- 최근 녹화 목록에서 프로젝트 재오픈 → 레시피 수정 → 재익스포트 가능.
- 자동 삭제 없음. 정리는 사용자가 폴더 단위로.

## 사용 동선

1. 메뉴바 상주 앱 (Dock 아이콘 없음). 패널에서 윈도우 선택 → 녹화 시작.
2. 글로벌 단축키 `⌥⌘R` 토글. 녹화 중 메뉴바 아이콘 = 빨간 점 + 경과 시간.
3. 카운트다운 없음. 앞부분 준비 동작은 앞 트리밍으로 제거.
4. 정지 → 에디터 창 자동 오픈: 미리보기 + 경량 편집 + 익스포트가 한 창에.
5. 익스포트 완료 → 파일이 클립보드에 자동 복사 + 알림 → PR/티켓에 ⌘V.
6. 에디터 창이 최근 녹화 목록의 진입점을 겸한다.
